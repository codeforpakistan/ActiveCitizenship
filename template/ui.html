User Interface Elements

There are various user interface elements contained in AIR, like navigation elements, different kind of buttons, with & without icons, different sizes, accordions & collapsibles, tabs & pills, stacked & regular, progress bars, sliders & many more.

Buttons

AIR comes with over 400 Premium Glyphicons as a font, that can be easely customized by color, size and even CSS3 attributes like text-shadow. The Glyphicons can be used not only as standalone, but in addition to other elements, like buttons.
